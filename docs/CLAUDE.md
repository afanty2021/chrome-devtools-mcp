# docs - 用户文档模块

## 模块职责

docs 模块包含 Chrome DevTools MCP 项目的用户文档，包括工具参考手册、故障排除指南和配置说明，为用户提供完整的使用指导。

## 文档结构

### 用户文档
- `tool-reference.md` - 完整的工具参考手册
- `troubleshooting.md` - 故障排除和常见问题解决

### 项目文档
- `README.md` - 项目介绍和快速开始指南
- `CHANGELOG.md` - 版本变更历史
- `CONTRIBUTING.md` - 贡献指南
- `SECURITY.md` - 安全政策
- `LICENSE` - 许可证文件

## 核心文档详解

### tool-reference.md
**功能**: 完整的 MCP 工具参考手册

**内容结构**:
- 按类别分组的工具列表
- 每个工具的详细说明
- 参数定义和使用示例
- 最佳实践和注意事项

**自动生成**:
- 从源代码工具定义自动生成
- 保持与代码同步更新
- 格式化的输出展示

### troubleshooting.md
**功能**: 故障排除指南

**主要章节**:
- 安装和配置问题
- 浏览器连接问题
- 网络和防火墙问题
- 性能优化建议
- 常见错误解决方案

### README.md
**功能**: 项目主文档和快速开始指南

**主要章节**:
- 项目介绍和特性
- 系统要求和安装
- MCP 客户端配置
- 工具列表和分类
- 配置选项说明
- 使用示例和最佳实践

## 文档生成机制
